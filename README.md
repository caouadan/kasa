# Notes about project kasa:

## Development

Install dependencies:
```
npm install
```

Start the development server:
```
npm run dev
```


Generate the production build:
```
npm run build
```

## Deployment on GitHub Pages
This project uses HashRouter, which ensures full compatibility with GitHub Pages (no 404 errors).

To deploy:
```
npm run deploy
```

After deployment, your site will be available at:
https://YOURUSERNAME.github.io/kasa/

Tech stack:
React
Vite
SCSS
